# Plan 02-01: Hero & Navigation Polish

## Objective

Refine the hero section and navigation bar to achieve pixel-perfect fidelity against the Design 2 and Design 4 wireframes. The hero currently has the right content but the layout needs significant restructuring: adding the signature dark green band behind "FOOTFALL LEASING", repositioning the tagline card, adding entry animations, and polishing the nav bar with scroll-aware glassmorphism.

## must_haves

### truths
- D-02: "FOOTFALL LEASING" sits on a **solid dark green band** (`#004337`) at the bottom of the hero
- D-03: The green band spans the full width and ~40% of the hero height
- D-04: The tagline card ("Brand Belongs") is positioned at the top-right of the green band, overlapping the background photo
- The hero background uses `ATMOSFIRE 01.jpeg` (current best available — PSD export deferred)
- Nav has "F" monogram logo left, links right, dark green background

### behaviors
- Hero text and tagline card animate in on page load (fade + slide)
- Nav gains a subtle backdrop-filter/shadow when user has scrolled past the hero
- Hamburger menu animates to X on open
- All transitions use the design system timing variables

### Task 4: Add Scroll-Reveal JavaScript

<action>
Create `src/components/scroll-reveal.js`:

```javascript
export function initScrollReveal() {
  const elements = document.querySelectorAll('.reveal');
  if (elements.length === 0) return;

  const observer = new IntersectionObserver(
    (entries) => {
      entries.forEach(entry => {
        if (entry.isIntersecting) {
          entry.target.classList.add('reveal--visible');
          observer.unobserve(entry.target); // Only animate once
        }
      });
    },
    { threshold: 0.15, rootMargin: '0px 0px -50px 0px' }
  );

  elements.forEach(el => observer.observe(el));
}
```

Update `src/main.js` to import and call `initScrollReveal()`.

### Task 5: Nav Scroll-Aware Glassmorphism

<read_first>
- src/styles/nav.css (current state)
- src/components/nav.js (current state)
</read_first>

<action>
Update `src/components/nav.js` to add scroll-aware behavior:

```javascript
// Inside initNav():
const nav = document.getElementById('main-nav');
if (nav) {
  window.addEventListener('scroll', () => {
    if (window.scrollY > 100) {
      nav.classList.add('nav--scrolled');
    } else {
      nav.classList.remove('nav--scrolled');
    }
  }, { passive: true });
}
```

Add to `src/styles/nav.css`:
```css
.nav--scrolled {
  background: rgba(0, 67, 55, 0.92);
  backdrop-filter: blur(12px);
  -webkit-backdrop-filter: blur(12px);
  box-shadow: var(--shadow-nav);
}
```

This makes the nav slightly transparent with blur when scrolled, giving a premium glassmorphism feel. On the hero (scroll position 0), the nav is solid green.
</action>
