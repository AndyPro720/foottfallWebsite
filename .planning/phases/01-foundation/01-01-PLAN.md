# Plan 01-01: Foundation & Design System

## Objective

Set up the Vite + Vanilla JS project from scratch. Define the complete CSS design system (colors, typography, spacing, breakpoints) as custom properties. Implement the global `scroll-snap` layout where each section occupies exactly `100vh` and snaps into place on scroll/swipe. Build the sticky navigation bar and skeleton section containers.

## must_haves

### truths
- D-01: Full-page scroll-snap sections (`scroll-snap-type: y mandatory` on `html`)
- D-02: Color palette — primary `#004337`, cream `#FAFAF5`, gold accent `#C9A96E`
- D-03: Typography — Playfair Display (serif) + Outfit (sans-serif) via Google Fonts
- D-04: 7 sections in order — Hero, Cities, Trade Areas, Categories, How We Work, About, Brands/Footer
- D-05: Mobile-first responsive with breakpoints at 480/768/1024/1440px

### behaviors
- Scrolling past a section boundary snaps cleanly to the next section
- Navigation links smooth-scroll to their target section
- Nav bar is sticky and remains visible during scroll
- Each section container fills the full viewport height
